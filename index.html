Portfolio

Dakerum

Website where you can leave an emoji after logging in via github. (NextJs, Prisma, TailwindCSS, Vercel, Clerk Auth)

City builder

University project of creating a city builder game from scratch. (Python) You can build houses, roads, factories, etc.

Diary

Angular application with a REST API backend. Backend server with Laravel. You can create an account and write diary entries. You can also see the entries of other users.

LMS

University Laravel project of creating a learning management system from scratch. With 2 roles (student and teacher) and different courses. Students can register and submit the tasks to courses and teachers can create courses, tasks and grade the assignments.

Light bulb game

JavaScript application where you can play a puzzle game with light bulbs, avoiding the obstacles. Aim of the game is to illuminate the whole map with proper placement of light bulbs.

Poll application

PHP application where you can create polls and vote on them. You can also see the results of the polls. You can also create an account to save your polls. You can also see the polls of other users.

Capitaly game

Simplified Capitaly game without any graphics written in Java. The game is played in the console. The aim of the game is to be the richest player. There are 3 types of players (greedy, tactical and careful). If a player has to pay, but he runs out of money because of this, he loses. In this case, his properties are lost, and become free to buy.

Tricky five-in-a-row

Variant of the well-known five-in-a-row game. The game is played on a (6x6, 10x10, 14x14) board. The aim of the game is to be the first to get 5 stones in a row.

Labyrinth

Labyrinth game in Java, where objective of the player is to escape from this labyrinth. The player starts at the bottom left corner of the labyrinth. He has to get to the top right corner of the labyrinth as fast he can, avoiding a meeting with the evil dragon. The player can move only in four directions: left, right, up or down. It is dark in the labyrinth, the player can see only the neighboring fields at a distance of 3 units.
